# Important Rules While Practicing

## 1. No AI During First Attempt
Do not use:
- ChatGPT
- Copilot
- AI autocomplete

during your first implementation attempt.

---

## 2. Use Timers
Practice under:
- 20 mins
- 30 mins
- 45 mins

This builds interview execution speed.

---

## 3. Focus On Completion
Do NOT chase perfect CSS initially.

Focus on:
- working logic
- rendering
- state management
- completion

---

## 4. Improve These Skills Specifically
- map()
- reduce()
- filter()
- nested rendering
- conditional rendering
- component composition
- state updates
- async handling
